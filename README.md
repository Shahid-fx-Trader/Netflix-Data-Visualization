# Netflix Data Visualization 📺

An interactive data analysis project that visualizes Netflix content trends, genre distributions, and performance metrics.

## Features
- 📊 Interactive data visualizations
- 📈 Genre and content analysis
- 🎬 Historical trend tracking
- 🐍 Built with Python and Jupyter Notebook

## Dataset
- `mymoviedb.csv` - Netflix movies/shows dataset

## Requirements
```python
pandas
numpy
matplotlib
seaborn
jupyter
plotly
```

## Installation
```bash
pip install pandas numpy matplotlib seaborn jupyter plotly
```

## Usage
1. Open Jupyter Notebook:
```bash
jupyter notebook "Netflix Analysis Data.ipynb"
```
2. Run all cells to generate visualizations
